📊 Dashboard Executivo de Vendas - Power BI
📌 Sobre o Projeto

Este projeto foi desenvolvido com o objetivo de analisar o desempenho comercial de uma empresa fictícia utilizando o conjunto de dados Sample Superstore.

O dashboard foi construído no Power BI e permite acompanhar indicadores estratégicos de vendas, lucro, pedidos e comportamento dos clientes por diferentes perspectivas.

🎯 Objetivos da Análise
-Monitorar o desempenho geral das vendas.
-Identificar regiões com melhor performance.
-Analisar categorias e segmentos mais lucrativos.
-Avaliar tendências de crescimento ao longo do tempo.
-Apoiar a tomada de decisões baseada em dados.

🛠️ Ferramentas Utilizadas
-Power BI Desktop
-Power Query
-DAX (Data Analysis Expressions)
-Microsoft Excel
-GitHub

📈 Indicadores Principais (KPIs)
-Receita Total
-Lucro Total
-Quantidade Vendida
-Total de Pedidos

📊 Páginas do Dashboard
1. Visão Executiva

Apresenta uma visão geral do negócio por meio dos principais indicadores e gráficos estratégicos:

-Receita por Região
-Receita por Categoria
-Receita por Segmento
-KPIs de desempenho
-Filtros interativos

2. Análise de Produtos

Permite identificar os produtos e categorias com melhor desempenho.

Principais análises:

-Top produtos por receita
-Top produtos por lucro
-Receita por categoria
-Lucro por categoria
-Quantidade vendida por produto
